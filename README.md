# Vendor Performance Analysis  
Data analytics project evaluating vendor and brand performance using Python, SQL, and Power BI.

## Overview  
This project analyzes sales, purchase, and profitability data to assess vendor performance, identify low‑performing brands, and highlight inventory and pricing inefficiencies. The analysis includes data preparation, exploratory analysis, statistical testing, and dashboard development.

## Key Components  
- Data extraction from SQL tables  
- Data cleaning and transformation in Python  
- Exploratory data analysis and visualizations  
- Vendor and brand performance assessment  
- Power BI dashboard for executive reporting  

## Technologies  
Python, Pandas, NumPy, Seaborn, Matplotlib  
SQLite  
Power BI  
Jupyter Notebook

## How to Run  
```
pip install -r requirements.txt
jupyter notebook Vendor_Performance_Analysis.ipynb
```

## Deliverables  
- Jupyter notebook with full analysis  
- Power BI dashboard  
- Visual assets (PNG)  
- Documentation
